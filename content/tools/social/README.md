## Content Types

| Type | Best For | Length |
|------|----------|--------|
| Single Tweet | Quick tips, milestones, links | ≤ 280 chars |
| Thread | Announcements, deep dives, tutorials | 3-10 tweets |
| Quote Tweet | Engagement, discussion starters | Context + reply |

## Posting Schedule

| Day | Content Type | Examples |
|-----|--------------|----------|
| Monday | Weekly tip | Tips/how-to content |
| Tuesday | Dev update | "This week we shipped..." |
| Wednesday | Community | Spotlight, appreciation |
| Thursday | Educational | Thread on a feature |
| Friday | Milestone/celebration | Stars, releases, wins |

## Thread Best Practices

1. **Hook tweet** (tweet 1): Grab attention with a bold claim or question
2. **Context** (tweets 2-3): Set up the problem or topic
3. **Value** (tweets 4-7): Core content, code snippets, insights
4. **CTA** (tweet 8+): Link to docs, ask for feedback, invite discussion

## Engagement Guidelines

- Reply to comments within 24 hours
- Like and retweet community content
- Use relevant hashtags: #AI #Agents #GoLang #DevTools
- Mention collaborators when appropriate
- Always credit sources and contributors
