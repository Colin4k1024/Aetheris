## Quick Reference

### X Post Types

| Type | Length | Use When |
|------|--------|----------|
| Single Tweet | ≤280 chars | Milestones, quick tips, links |
| Thread | 3-10 tweets | Announcements, tutorials, deep dives |
| Quote Tweet | Variable | Engaging with others, adding context |

### Article Length

| Platform | Recommended | Max |
|----------|------------|-----|
| 掘金 | 1500-3000字 | 5000字 |
| 知乎 | 2000-4000字 | 8000字 |
| Newsletter | 500-800字 | 1200字 |

### Posting Frequency

| Channel | Frequency |
|---------|-----------|
| X (Twitter) | 3-5 posts/week |
| 掘金 | 1-2 posts/week |
| 知乎 | 1 post/2 weeks |
| Newsletter | 1-2/month |

## Content Pillars

1. **Technical Education** - How-tos, deep dives, architecture
2. **Product Updates** - Releases, features, improvements
3. **Community** - Contributors, users, use cases
4. **Industry Perspective** - Agent patterns, durability, production AI

## Getting Started

1. Choose the appropriate template for your content type
2. Fill in the placeholders with your content
3. Review against the posting guidelines
4. For social media, schedule using Buffer/Later
5. For articles, proofread and add images before publishing
