# 中文内容模板

本目录包含用于掘金(Juejin)和知乎(Zhihu)平台的文章模板和内容规划。

## 目录结构

```
content/templates/
├── README.md              # 本文件
├── calendar.md            # 月度文章日历
├── content-calendar.md    # 两个月内容计划
├── article-template-zhihu.md   # 知乎文章模板
├── article-template-juejin.md  # 掘金文章模板
├── seo-guidelines.md      # SEO优化指南
└── template-overview.md   # 模板概述
```

## 平台特点

### 掘金 (Juejin)
- 面向开发者社区
- 技术文章为主
- 代码片段支持良好
- 有推荐算法
- 支持系列文章

### 知乎
- 更广泛的知识社区
- 问答形式流行
- 深度分析文章受欢迎
- 适合技术趋势讨论

## 写作风格

1. **开头**: 吸引注意力的hook，1-2句话
2. **背景**: 为什么这个话题重要
3. **正文**: 分层清晰，每个章节有明确主题
4. **代码**: 保持简洁，突出重点
5. **结尾**: 总结+行动号召
