## 建议的社交分享图设计

### 尺寸要求
- **Twitter/X**: 1200 x 675 像素 (2:1)
- **LinkedIn**: 1200 x 627 像素 (1.91:1)
- **Facebook**: 1200 x 630 像素

- 背景: #0F172A (深空黑)

- 文字: #FFFFFF (白色)

- [Canva](https://canva.com) - 模板丰富
