## About Aetheris

**Aetheris** (also known as CoRag) is an execution runtime for intelligent agents — "Temporal for Agents." It provides a durable, replayable, and observable environment where AI agents can plan, execute, pause, resume, and recover long-running tasks.

### One-Liner

> Aetheris is a durable execution runtime for AI agents — think Temporal for AI agents with built-in observability and enterprise features.

### Tagline

**"Temporal for Agents"** — Durable, replayable, and observable AI agent execution.

## Features

| Feature | Description |
|---------|-------------|
| **Durable Execution** | Job state persists to PostgreSQL |
| **At-Most-Once** | Effects Ledger prevents duplicate execution |
| **Human-in-the-Loop** | Pause and wait for human approval |
| **Trace UI** | Built-in execution visualization |
| **Evidence Chain** | Complete audit trail with export |
| **RBAC** | Role-based access control |
| **Multi-Region** | Regional scheduling support |
