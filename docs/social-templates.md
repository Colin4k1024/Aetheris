## Blog Announcement

```markdown
# Introducing Aetheris: Durable Execution for AI Agents

We're excited to announce Aetheris — an execution runtime that brings durable, replayable, and observable execution to AI agents.

## The Problem

Building AI agents is hard. Making them reliable in production is even harder:

- **State loss**: Agent crashes mid-execution, lose all progress
- **Duplicate execution**: Network timeout causes retry, tool called twice
- **No observability**: Can't see what the agent actually did
- **Compliance gaps**: Can't prove agent decisions for audit

## Enter Aetheris

Aetheris treats agents as virtual processes with:

1. **Durable Execution** — Job state persists to PostgreSQL
2. **At-Most-Once** — Effects Ledger prevents duplicate calls
3. **Trace UI** — Built-in execution visualization
4. **Evidence Chain** — Complete audit trail with export

## Get Started

```bash
git clone https://github.com/Colin4k1024/Aetheris
cd Aetheris
