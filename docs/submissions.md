## Submit to awesome-go

### Prerequisites

- Working Go project with proper README
- At least 100 GitHub stars (recommended)
- Active development

### Submission URL

https://github.com/avelino/awesome-go/blob/main/README.md

### How to Submit

1. Fork the repo: https://github.com/avelino/awesome-go
2. Add your project to the appropriate category
3. Create a PR

### Recommended Category

```
## Agent Frameworks

- [Aetheris](https://github.com/Colin4k1024/Aetheris) - Durable execution runtime for AI agents - "Temporal for Agents"
```

### PR Template

```markdown
## Project Name: Aetheris

### Repository URL
https://github.com/Colin4k1024/Aetheris

### Category
Agent Frameworks / AI Development Tools

### Why it should be added
Aetheris is a durable execution runtime for AI agents, providing:
- Persistent job state (survives crashes)
- At-Most-Once execution guarantee
- Built-in Trace UI
- Enterprise features (RBAC, audit)

### Additional Info
- License: Apache 2.0
- Stars: [current count]
- Last commit: [date]
```
